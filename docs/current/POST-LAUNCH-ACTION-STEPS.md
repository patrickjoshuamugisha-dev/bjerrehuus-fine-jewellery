## Accounting Sync: Shopify to e-conomic

**Status:** Phase two setup if Vibe/bookkeeper wants automated accounting sync.

**Recommendation:** Use Storebuddy for the Shopify to e-conomic accounting sync.

Why:
- Vibe uses e-conomic as her accounting system.
- Storebuddy is listed by e-conomic as the Shopify integration partner.
- Storebuddy can sync orders, credit notes/refunds, payments, fees, payouts, and reconciliation data.
- This avoids custom development.

Not a simple one-click task:
- Requires Vibe's e-conomic access.
- Requires Shopify admin access.
- Requires a Storebuddy subscription/trial setup.
- Should be reviewed with Vibe's bookkeeper/accountant before automation is turned on.
- Should be tested with a real Shopify order and refund after payments are live.

Default next step after launch:
1. Ask Vibe whether her bookkeeper/accountant wants Shopify orders synced automatically into e-conomic.
2. If yes, install Storebuddy from the Shopify App Store or add it via e-conomic Market.
3. Connect Storebuddy to the Bjerrehuus Shopify store.
4. Connect Storebuddy to Vibe's e-conomic agreement.
5. Confirm accounting defaults with the bookkeeper/accountant before activation.
6. Run one paid-order sync test and one refund/credit-note sync test.

Client-facing wording:

> Shopify kan kobles til e-conomic via Storebuddy, så ordrer, betalinger, gebyrer, refunds og udbetalinger bogføres automatisk. Det kræver adgang til både Shopify og e-conomic samt valg af regnskabsopsætning, så vi anbefaler at tage det med din bogholder/revisor efter launch. Hvis det skal være aktivt fra første ordre, skal vi sætte en separat opsætningssession op før launch.

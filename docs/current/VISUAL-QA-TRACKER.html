Theme editor

Theme code

1. Capture

One row per issue. Use a screenshot for layout/image notes.

2. Classify

Choose Shopify admin, theme editor, or theme code.

3. Batch

4. Fix

Patch the dev Dawn theme only.

5. Verify

Check desktop/mobile and run Theme Check.
